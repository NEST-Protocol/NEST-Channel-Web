## Help

### To developer

1. Create environments file.
   ```shell
   cp .env.example .env
   ```
2. Take your [REACT_APP_INFURA_KEY](https://infura.io/)、[REACT_APP_ETHERSCAN_KEY](https://etherscan.io/)、[REACT_APP_BSCSCAN_KEY](https://bscscan.com/).

3. Install dependence.

   ```shell
   yarn
   ```

4. Serve the web.
   ```shell
   yarn start
   ```
